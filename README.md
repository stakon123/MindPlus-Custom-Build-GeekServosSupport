# MindPlus-Custom-Build-GeekServosSupport
A Custom Build of Mind Plus for S1 Kit

Οι αλλαγές έγιναν πάνω στο extension του vmichas https://github.com/vmihas/MindPlus-Custom-Build

Προστέθηκαν οι εντολές ελέγχου των GeekServo Πράσινου και Γκρι <img width="281" height="248" alt="image" src="https://github.com/user-attachments/assets/f5a348a5-796b-4010-bc35-3b89981df916" />

Τόσο σε Online όσο και Offline mode.

<img width="499" height="140" alt="image" src="https://github.com/user-attachments/assets/f8bd81b9-e58f-401c-94cc-861504e94630" />


Για **εγκατάσταση** αφού πρώτα εγκατασταθεί το **custom build του vmihas** πρέπει να αλλαχτεί το αρχείο **app.asar** στο C:\Program Files (x86)\Mind+\resources με αυτό που υπάρχει στα Releases


Ουσιαστικά οι αλλαγές έγιναν στα αρχεία polytechs1.chunk.js και lib.min.js

Τα νέα αρχεία καθώς και μόνο οι αλλαγές φαίνονται στα αρχεία του Repository
