# MindPlus-Custom-Build-GeekServosSupport
A Custom Build of Mind Plus for S1 Kit

Οι αλλαγες εγιναν πανω στο extension του vmichas https://github.com/vmihas/MindPlus-Custom-Build

Προσθεθηκαν οι εντολές ελεγχου των GeekServo Πράσινου και Γκρι <img width="281" height="248" alt="image" src="https://github.com/user-attachments/assets/f5a348a5-796b-4010-bc35-3b89981df916" />

Τόσο σε Online οσο και Offline mode.

<img width="499" height="140" alt="image" src="https://github.com/user-attachments/assets/f8bd81b9-e58f-401c-94cc-861504e94630" />


Για **εγκατασταση** αφου πρωτα εγκατασταθει το **custom build του vmihas** πρεπει να αλλαχτει το αρχειο **app.asar** στο C:\Program Files (x86)\Mind+\resources
